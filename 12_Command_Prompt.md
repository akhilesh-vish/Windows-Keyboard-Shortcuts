# 💻 Command Prompt (CMD)

Command Prompt, commonly referred to as **CMD**, is Windows' classic command-line interpreter. It provides a fast and efficient way to perform administrative tasks, automate repetitive processes, troubleshoot system issues, and execute commands that may be difficult or impossible through the graphical interface.

Although newer tools such as PowerShell and Windows Terminal offer additional capabilities, Command Prompt remains an essential utility for Windows users, developers, and IT professionals.

> [!TIP]
> Even if you primarily use the graphical interface, learning a few Command Prompt shortcuts can make troubleshooting and system administration much faster.

# ⌨️ Keyboard Shortcuts

| Shortcut           | Description                                                                                        | Availability    |
| ------------------ | -------------------------------------------------------------------------------------------------- | --------------- |
| **Ctrl + C**       | Cancel the currently running command or process. If no command is running, copy the selected text. | Windows 10 / 11 |
| **Ctrl + V**       | Paste text from the clipboard into the Command Prompt window.                                      | Windows 10 / 11 |
| **Ctrl + A**       | Select all text currently displayed in the Command Prompt window.                                  | Windows 10 / 11 |
| **Ctrl + M**       | Enable Mark Mode for selecting text using the keyboard.                                            | Windows 10 / 11 |
| **Ctrl + F**       | Search the Command Prompt output.                                                                  | Windows 10 / 11 |
| **Ctrl + Home**    | Scroll to the beginning of the command history (when command line is empty).                       | Windows 10 / 11 |
| **Ctrl + End**     | Scroll to the end of the command history (when command line is empty).                             | Windows 10 / 11 |
| **↑ (Up Arrow)**   | Recall the previous command from command history.                                                  | Windows 10 / 11 |
| **↓ (Down Arrow)** | Move to the next command in the command history.                                                   | Windows 10 / 11 |
| **F7**             | Display a numbered list of previously executed commands.                                           | Windows 10 / 11 |
| **F8**             | Search command history based on the current input.                                                 | Windows 10 / 11 |
| **F9**             | Run a previous command by entering its number from the F7 history list.                            | Windows 10 / 11 |
| **Tab**            | Auto-complete file and folder names.                                                               | Windows 10 / 11 |
| **Shift + Tab**    | Cycle backward through auto-complete suggestions.                                                  | Windows 10 / 11 |
| **Ctrl + ← / →**   | Move the cursor one word at a time.                                                                | Windows 10 / 11 |
| **Home**           | Move the cursor to the beginning of the current command.                                           | Windows 10 / 11 |
| **End**            | Move the cursor to the end of the current command.                                                 | Windows 10 / 11 |
| **Esc**            | Clear the current command line without executing it.                                               | Windows 10 / 11 |
| **Alt + Enter**    | Toggle Full Screen mode (supported on older systems; limited on modern Windows versions).          | Windows 10 / 11 |
| **Alt + F4**       | Close the Command Prompt window.                                                                   | Windows 10 / 11 |

# ⭐ Most Useful Shortcuts

These shortcuts are the ones you'll use most often while working in Command Prompt.

| Shortcut             | Why It's Useful                                                   |
| -------------------- | ----------------------------------------------------------------- |
| **↑ / ↓ Arrow Keys** | Quickly browse previously executed commands.                      |
| **Tab**              | Auto-complete long file and folder names, reducing typing errors. |
| **Ctrl + C**         | Stop a running command immediately.                               |
| **Ctrl + V**         | Paste long commands without retyping them.                        |
| **F7**               | Display your recent command history in a convenient list.         |
| **F8**               | Search through command history to reuse previous commands.        |
| **Ctrl + ← / →**     | Move through long commands one word at a time.                    |
| **Esc**              | Clear the current command line instantly.                         |

# 💡 Productivity Tips

* Use the **Tab** key whenever typing long file paths or filenames. It saves time and reduces typing mistakes.
* Press **↑** to reuse recently executed commands instead of typing them again.
* Use **F7** to display your command history and quickly repeat previous commands.
* Copy commands from documentation or websites and paste them using **Ctrl + V**.
* If a command appears to be stuck or takes too long, press **Ctrl + C** to safely interrupt it.
* Learn common navigation shortcuts such as **Home**, **End**, and **Ctrl + Arrow Keys** when editing lengthy commands.

# 📝 Notes

> [!NOTE]
> Modern versions of Windows support standard keyboard shortcuts such as **Ctrl + C** and **Ctrl + V** directly in Command Prompt.

> [!NOTE]
> Some legacy shortcuts may behave differently depending on the version of Windows or the console host being used.

> [!NOTE]
> Command Prompt supports command history only for the current session unless additional tools or scripts are used.

> [!NOTE]
> Many administrative commands require Command Prompt to be launched with **Run as administrator**.

# ⚙️ Requirements

| Requirement                  | Details                                                                 |
| ---------------------------- | ----------------------------------------------------------------------- |
| **Operating System**         | Windows 10 or Windows 11                                                |
| **Administrator Privileges** | Required for many system administration commands.                       |
| **Command Extensions**       | Enabled by default in modern versions of Windows.                       |
| **QuickEdit Mode**           | Enabled by default, allowing mouse selection and clipboard operations.  |
| **Console Host**             | Some keyboard behavior may differ when running inside Windows Terminal. |

# 💭 Did You Know?

> 💡 **Command Prompt remembers your command history.**
> Press **F7** to display a list of recently executed commands, making it easy to rerun frequently used commands.

---

> 💡 **The Tab key is one of the biggest time savers.**
> Instead of typing long file or folder names, press **Tab** to let Command Prompt complete them automatically.

---

> 💡 **You can interrupt most running commands.**
> If a command hangs or takes longer than expected, press **Ctrl + C** to stop it without closing the Command Prompt window.

---

> 💡 **Command Prompt is still widely used.**
> Even though PowerShell and Windows Terminal offer more advanced features, many Windows utilities, installation scripts, and troubleshooting guides continue to rely on Command Prompt.
