# 🔔 Notifications & Quick Settings

Notifications and Quick Settings give you instant access to important alerts and commonly used system controls without opening the Settings app. From adjusting your Wi-Fi and Bluetooth connections to enabling Focus Assist, changing volume levels, or reviewing recent notifications, these features help you stay productive while keeping interruptions under control.

Learning these keyboard shortcuts allows you to access essential system controls in seconds, making everyday tasks faster and more efficient.

> [!TIP]
> If you regularly work in meetings or while studying, combine **Quick Settings** with **Do Not Disturb (Focus Assist)** to minimize distractions and stay focused.

# ⌨️ Keyboard Shortcuts

| Shortcut        | Description                                                                              | Availability    |
| --------------- | ---------------------------------------------------------------------------------------- | --------------- |
| **Win + A**     | Open Quick Settings (Windows 11) or Action Center (Windows 10).                          | Windows 10 / 11 |
| **Win + N**     | Open Notification Center and Calendar.                                                   | Windows 11      |
| **Win + B**     | Move keyboard focus to the notification area (System Tray).                              | Windows 10 / 11 |
| **Win + K**     | Open the Connect panel to pair or connect wireless displays and supported audio devices. | Windows 10 / 11 |
| **Win + P**     | Open the Project menu to switch display modes such as Duplicate or Extend.               | Windows 10 / 11 |
| **Win + I**     | Open the Settings app.                                                                   | Windows 10 / 11 |
| **Win + U**     | Open Accessibility settings.                                                             | Windows 10 / 11 |
| **Win + Space** | Switch between installed keyboard layouts and input languages.                           | Windows 10 / 11 |
| **Win + L**     | Lock your computer immediately.                                                          | Windows 10 / 11 |
| **Esc**         | Close Quick Settings, Notification Center, or any open flyout panel.                     | Windows 10 / 11 |
| **Tab**         | Move between controls within Quick Settings or Notification Center.                      | Windows 10 / 11 |
| **Shift + Tab** | Move to the previous control.                                                            | Windows 10 / 11 |
| **Arrow Keys**  | Navigate through Quick Settings buttons and notification cards.                          | Windows 10 / 11 |
| **Enter**       | Activate the selected Quick Settings control or open the selected notification.          | Windows 10 / 11 |
| **Spacebar**    | Toggle the selected Quick Settings button when supported.                                | Windows 10 / 11 |

# ⭐ Most Useful Shortcuts

These shortcuts provide the quickest access to Windows notifications and system controls.

| Shortcut    | Why It's Useful                                                                                |
| ----------- | ---------------------------------------------------------------------------------------------- |
| **Win + A** | Quickly access Wi-Fi, Bluetooth, volume, brightness, airplane mode, and other system controls. |
| **Win + N** | View recent notifications without interrupting your work.                                      |
| **Win + K** | Connect to wireless displays, TVs, or Bluetooth audio devices.                                 |
| **Win + P** | Instantly switch between display modes when using external monitors or projectors.             |
| **Win + I** | Open Settings to make more advanced system changes.                                            |
| **Win + B** | Navigate the System Tray entirely from the keyboard.                                           |
| **Win + L** | Lock your PC before stepping away from your desk.                                              |

# 💡 Productivity Tips

* Use **Win + A** instead of opening the Settings app when you only need to adjust common settings like Wi-Fi, Bluetooth, or brightness.
* Press **Win + N** occasionally to review notifications instead of letting them interrupt your workflow.
* Learn **Win + P** before giving presentations—it makes switching between laptop and projector displays much faster.
* Use **Win + K** to connect to wireless displays without navigating through multiple Settings menus.
* Navigate Quick Settings using **Tab**, **Arrow Keys**, and **Enter** if you prefer using the keyboard instead of the mouse.
* Lock your PC with **Win + L** whenever you leave your workstation, even for a short time.

# 📝 Notes

> [!NOTE]
> In **Windows 11**, Quick Settings and Notification Center are separate panels accessed with **Win + A** and **Win + N**.

> [!NOTE]
> In **Windows 10**, many of these controls are grouped together in the Action Center.

> [!NOTE]
> The available Quick Settings buttons depend on your hardware. For example, devices without Bluetooth or Wi-Fi won't display those controls.

> [!NOTE]
> Some Quick Settings features, such as wireless display connections, require compatible hardware.

# 💭 Did You Know?

> 💡 **Quick Settings can be customized.**
> In Windows 11, you can add, remove, and rearrange Quick Settings buttons so your most-used controls are always within easy reach.

---

> 💡 **Notifications remain available until you dismiss them.**
> Even if you miss a pop-up notification, you can review it later by opening the Notification Center.

---

> 💡 **The System Tray is fully keyboard accessible.**
> Press **Win + B** to move focus to the notification area, then use the **Arrow Keys** to navigate through system icons such as volume, network, battery, and security.

---

> 💡 **Quick Settings aren't just for laptops.**
> Desktop users can quickly adjust network settings, audio output, accessibility features, display projection, and connected devices without opening the full Settings app.
