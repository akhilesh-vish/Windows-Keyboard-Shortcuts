# 🖥️ Virtual Desktops

Virtual Desktops are one of Windows' most powerful multitasking features, allowing you to create separate workspaces for different activities without the need for additional monitors. Instead of cluttering a single desktop with dozens of open windows, you can organize your applications into dedicated desktops for work, study, gaming, entertainment, or personal tasks.

By learning these shortcuts, you can switch between desktops instantly, stay organized, and reduce distractions while working.

> [!TIP]
> Think of each Virtual Desktop as a separate workspace. For example, keep work-related applications on one desktop, communication apps on another, and entertainment apps on a third.

# ⌨️ Keyboard Shortcuts

| Shortcut            | Description                                                                                                                                | Availability    |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ | --------------- |
| **Win + Tab**       | Open Task View to display all open windows and virtual desktops.                                                                           | Windows 10 / 11 |
| **Win + Ctrl + D**  | Create a new virtual desktop and switch to it immediately.                                                                                 | Windows 10 / 11 |
| **Win + Ctrl + ←**  | Switch to the virtual desktop on the left.                                                                                                 | Windows 10 / 11 |
| **Win + Ctrl + →**  | Switch to the virtual desktop on the right.                                                                                                | Windows 10 / 11 |
| **Win + Ctrl + F4** | Close the current virtual desktop. Open applications remain running and move to the previous desktop.                                      | Windows 10 / 11 |
| **Alt + Tab**       | Switch between open applications on the current desktop. Depending on your Task View settings, it can also show windows from all desktops. | Windows 10 / 11 |
| **Win + D**         | Show or hide the desktop on the current virtual desktop.                                                                                   | Windows 10 / 11 |
| **Win + M**         | Minimize all open windows on the current desktop.                                                                                          | Windows 10 / 11 |
| **Win + Shift + M** | Restore windows minimized using **Win + M**.                                                                                               | Windows 10 / 11 |
| **Win + ← / →**     | Snap the active window to the left or right side of the current desktop.                                                                   | Windows 10 / 11 |
| **Win + ↑ / ↓**     | Maximize, restore, or minimize the active window.                                                                                          | Windows 10 / 11 |

# 💡 Productivity Tips

* Create a dedicated **Work Desktop** for productivity applications such as Microsoft Office, Visual Studio, or web browsers used for work.
* Keep messaging applications like Teams or Discord on a separate desktop to reduce distractions while focusing.
* Use another desktop for entertainment, such as music players, streaming services, or gaming applications.
* If you're researching a topic, dedicate one desktop to reference material and another to writing or coding.
* Before closing a virtual desktop, don't worry about losing your work—Windows automatically moves open applications to the previous desktop.
* Combine **Virtual Desktops** with **Snap Layouts** to create highly organized workspaces on each desktop.

# 📝 Notes

> [!NOTE]
> Virtual Desktops do **not** create separate user sessions. All desktops share the same running applications and system resources.

> [!NOTE]
> Closing a virtual desktop does **not** close the applications running on it. Windows automatically transfers them to another desktop.

> [!NOTE]
> In Windows 11, each virtual desktop can have its own desktop wallpaper, making it easier to identify different workspaces.

> [!NOTE]
> The behavior of **Alt + Tab** can be customized in **Settings → System → Multitasking**, allowing you to choose whether it shows windows from the current desktop or all desktops.

# ⚙️ Requirements

| Requirement                    | Details                                                                                       |
| ------------------------------ | --------------------------------------------------------------------------------------------- |
| **Operating System**           | Windows 10 or Windows 11                                                                      |
| **Task View**                  | Enabled by default in Windows.                                                                |
| **Multiple Monitors**          | Not required. Virtual Desktops work on both single- and multi-monitor setups.                 |
| **Windows 11 Personalization** | Different wallpapers for each virtual desktop are available only in Windows 11.               |
| **System Resources**           | Running many applications across multiple desktops still uses your computer's memory and CPU. |

# 💭 Did You Know?

> 💡 **There is no fixed limit to the number of Virtual Desktops you can create.**
> The practical limit depends on your computer's available memory and system performance.

---

> 💡 **Each desktop remembers its own open windows.**
> You can leave applications exactly where they are and return to the same workspace later without reorganizing your windows.

---

> 💡 **Virtual Desktops don't improve performance.**
> They're designed to improve organization, not reduce memory usage. Applications continue running even when they're on another desktop.

---

> 💡 **Windows 11 lets you personalize each desktop.**
> Assign different wallpapers to your Work, Study, Gaming, or Personal desktops to quickly identify which workspace you're using.
