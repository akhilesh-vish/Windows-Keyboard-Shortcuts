# 🖥️ Essential Windows Shortcuts

Master the most commonly used Windows keyboard shortcuts to navigate your PC faster, improve productivity, and reduce reliance on the mouse. Whether you're a beginner or an experienced user, these essential shortcuts will help you work more efficiently every day.

> [!TIP]
> Start by learning a few shortcuts at a time. Once they become second nature, gradually add more to your workflow.

# 💡 Productivity Tips

- **Learn shortcuts gradually.** Focus on mastering three to five new shortcuts each week instead of trying to memorize them all at once.
- **Use `Alt + Tab` for multitasking.** It's much faster than clicking between open windows with your mouse.
- **Save your work often.** Press `Ctrl + S` regularly, especially when working on important documents or projects.
- **Take advantage of Clipboard History.** Enable `Win + V` to keep multiple copied items available instead of only the most recent one.
- **Use `Win + R` for speed.** You can launch tools like `cmd`, `powershell`, `notepad`, `control`, and many others directly from the Run dialog.
- **Lock your PC when stepping away.** `Win + L` helps protect your personal information and prevents unauthorized access.

# ⚙️ Requirements

| Requirement | Details |
|------------|---------|
| **Operating System** | Windows 10 or Windows 11 |
| **Keyboard** | A standard keyboard with a **Windows (⊞ Win)** key |
| **Clipboard History** | Enable Clipboard History to use **Win + V** |
| **Voice Typing** | A microphone and supported language pack are required for **Win + H** |
| **Xbox Game Bar** | Must be installed and enabled to use **Win + G** |
| **Wireless Display** | Compatible hardware is required for **Win + K** |

# 💭 Did You Know?

> 💡 **Peek at your desktop without minimizing anything.**  
> Hold **Win + , (Comma)** to temporarily make all open windows transparent. Release the keys, and everything returns exactly as it was.

> 💡 **Clipboard History can store more than one item.**  
> Once enabled, **Win + V** lets you access multiple copied items instead of only the most recent one.

> 💡 **The Run dialog is more powerful than many users realize.**  
> With **Win + R**, you can quickly launch built-in Windows tools, applications, folders, and administrative utilities without searching through menus.
